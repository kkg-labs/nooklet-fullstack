### Setup

1. Copy env template and configure values:
```

```

2. Install dependencies:
```

```

3. Start Postgres (Docker):
```
cd nooklet-db && docker-compose up -d
```

4. Run dev server (HMR + Inertia + Vite):
```
npm run dev
```
